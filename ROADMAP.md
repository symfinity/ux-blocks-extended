## Details

### 0.1.x — Foundation

Compound Twig components, registry-aligned markup, package role CSS, Flex recipe

### 0.2.x — Navigation and overlays

Accordion, Dialog, Drawer, Dropdown Menu, Popover, Tabs, Tooltip, native interaction polish

### 0.3.x — Composition patterns

Card variants, Description List, Empty State, Hero, List Groups, Panel, Split View, Stepper, Wizard foundations

### 0.4.x — Application layouts

App Shell, Dashboard Shell, Sidebar layouts, Topbar layouts, Workspace layouts, responsive composition helpers

### 0.5.x — Data presentation

Data List, Data Grid, Key Value layouts, Metric cards, Stat groups, Timeline, Activity feed

### 0.6.x — Content blocks

Callout, Feature Grid, FAQ, Pricing, Testimonial, Team, Logo Cloud, Media layouts

### 0.7.x — Accessibility polish

Keyboard navigation, focus management, ARIA refinements, reduced motion support, RTL validation, browser compatibility

### 0.8.x — API consolidation

Component consistency, slot alignment, variant normalization, registry refinement, documentation improvements

### 0.9.x — Release candidate

Deprecation cleanup, migration guides, cookbook, test matrix, API freeze

### 1.0.x — Stable API

Semver-stable component contracts, stable registry, stable Twig API, long-term maintenance policy
